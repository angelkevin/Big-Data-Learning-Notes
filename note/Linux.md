# Linux

## 目录与路径

```shell
.	#代表此层目录
..	#代表上一层目录
-	#代表前一个工作目录
~	#代表目前用户身份所在的主文件夹
~account	#代表account这个用户的主文件夹
cd	#切换目录
pwd	#显示当前目录
mkdir	#新建一个新目录
rmdir	#删除一个空目录
ls	#查看文件与目录
	-a	#全部的文件
	-d	#仅列出目录本身
cp  #复制文件或者目录
	-r	#持续递归复制
rm	#删除文件
	-r	#递归删除
	-i	#互动模式
mv	#移动文件与目录或者更名
basename	#获得路径最后的文件名
dirname	#获取目录名
cat	#直接查看文件内容
	-A	#可以列出一些特殊字符
	-n	#打印行号
tac	#反向列示
```
